# 💻 CareerForge Client Application

This directory contains the React (Vite) and TailwindCSS frontend application that powers the interactive workspaces of the CareerForge AI Career & Resume Suite.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

For full installation guidelines, styling protocols, and component blueprints, please refer to the primary [Root README.md](../README.md).
